Pipes números

Pipes incluidos en Angular - Usualmente en el Common Module para manejar números.

  • {{ netSales }} --> {{ netSales | number }}
  • {{ netSales }} --> {{ netSales | number: "1.2-2" }}
  • {{ netSales }} --> {{ netSales + 2000000 | currency: "EUR":"symbol-narrow" }}
  • {{ netSales }} --> {{ netSales + 2000000 | currency: "EUR":"symbol-narrow":"1.0-4" }}
  • {{ percentage }} --> {{ percentage | percent: '2.2-2' }}
  • {{ percentage }} --> {{ percentage | percent }}
